## Misc

To run the code on your machine just download the notebook.

> [!WARNING]  
> For the TabPFN models to load, an api token is needed. The following `.env` file should be present in your working directory:
> 
> `TABPFN_TOKEN='<your-token>'`
>
> (Any other way of injecting this environment variable will also work).
